 Help: Projects

New project: Give a name, describe your project, add a folder name, select image or an environment, and define its scope.

List project: Use the search bar on the top of the page to filter them.
indicates that this project is not associated to any environment yet.

In the small dropdown menu with the buttons you can access all environments to which the project is bound to, stop or restart them.

At the bottom of the dropdown panel you can hide, configure or delete the Project.

Configure Project - Meta: update project name, change its visibility among Kooplex users or update the description.

Configure Project - Collaboration: Start typing the name or username of the collaborators you'd like to add to the project. Optionally give them administrator role. Also you can choose a default environment for the collaborators.

Configure Project - Environment: Attach this project to your environments.

Join project: In this page, you can join public projects. Use the switch to select the project you want to join and click on the Collaborate button. Toggle the image button if you would like to use the same environment for the chosen project.

Show/hide project: You have the option to hide or reveal the projects on the List Projects page. Use the toggle button to select the projects andd click the "Save al changes" button to finalize your choice.
