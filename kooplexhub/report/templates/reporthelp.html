Help: Reports

Welcome to Report. In this section, we have two navigation options. Let's explore them over!

List of reports: In this page, the reports are disposed of by card format. To search for a specific report, use the search bar on the top of the page.

On the top of the card, right next to the name, you can click the green button to open report. On the bottom, we can press the tool button to configure the report and the trash bin button to delete it.

New report: Create a new report. Project: associate the report to a project The name of the report: Name the report Folder: Associate the report to an existent folder Description: write some description to your report. Static: check if the report is static or not. Scope: Select the report status (choose between public, internal or private) Image: Associate the report to an image Index: name some index

For every non-static reporttypes, such as shiny, plotly etc a separate environment is prepared, which can be accessed under List report environments .
