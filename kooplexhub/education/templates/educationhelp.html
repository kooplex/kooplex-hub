Help: Education

Student course: List all of your attended courses. Use the search bar on the top of the page to filter.

Student assignments: List of all the assignments of your attended courses. Use this page to submit assignments or check your scores and the given feedbacks.

Course management: List of all the courses where you are a teacher. Use the search bar on the top of the page to filter.

In the small dropdown menu with the buttons you can access all environments to which the course is bound to, stop or restart them.

On the bottom part of the dropdown panel use the configure button to update course details!

Course configuration: Update the name, description or the default image of the course

Course configuration - Teacher: Start typing the name or the username who you'd like to add as a teacher to the course. Click on their name and Save the changes

Course configuration - Student: Start typing the name or the username who you'd like to add as a student to the course. Click on their name and Save the changes

Course configuration - Groups: Organize students into groups. You can hand out different assignment to each group.

Course configuration - Group students: Once a group is created it can be populated with students.

Assignment management - New: Create new assignment.

Assignment management - Configure: Update name, description, handout and collection time of the task and the size limit for each submission. Optionally the folder can be automatically deleted after submission (Erase button).

Assignment management - Mass assignment: Manage handout, collection and correction of an assignment, or reassign it.

Assignment management - Handle assignment: Select a course in the dropdown menu and handout, collect or correct (give score and feedback) for an assignment, or reassign it individually. Use filter by typing in the search bar on the top of this table.

Assignment management - Student points: Results of the students for a given course.
